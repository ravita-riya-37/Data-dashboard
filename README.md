# Data-dashboard

Data Visualization & Calculation Dashboard
Overview

This project is a comprehensive dashboard tool designed to visualize data, perform calculations, and provide an interactive summary. It allows users to build charts, calculate statistics, and display meaningful insights from large datasets.

Features
1. Chart Building

Create dynamic and interactive charts: Line, Bar, Pie, Area.

Customize chart types and view data with labels and tooltips.

Save charts as JPG images for reporting or presentations.

2. Calculation Tool

Calculate minimum, maximum, and total values from JSON data.

Perform comparisons: greater than, less than, or equal to.

Retrieve corresponding datasets based on calculation results.

3. Summary Dashboard

Display data summaries and visualizations in an intuitive interface.

Designed specifically for Blackcoffer projects with large datasets.

Recent Enhancements

Developed an interactive dashboard using the MERN stack and advanced chart libraries.

Implemented fast filtering and sorting of 1,000+ MongoDB entries using Recharts and Chart.js.

Optimized search operations to complete in milliseconds, improving efficiency.

Built a RESTful API integrated with React for seamless data operations using MongoDB aggregation.

About

Data-dasgboard is a MERN-based dashboard for creating charts, saving them as images, performing calculations, and interacting with data efficiently. It is designed for visualizing large datasets, such as 1,000+ MongoDB entries, and was built for Blackcoffer assignment projects.

Tech Stack:

MERN (MongoDB, Express, React, Node.js)

Chart.js, Recharts

REST APIs, CRUD operations
